# llm-finetune

Single-header C++17 library for OpenAI fine-tuning job lifecycle management.

## Structure
- `include/llm_finetune.hpp` — single-header implementation
- `examples/` — usage examples
- `CMakeLists.txt` — cmake build (requires vcpkg curl where applicable)

## Build
```bash
cmake -B build && cmake --build build
```
